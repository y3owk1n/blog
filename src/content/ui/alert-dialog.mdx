## Installation

1. Install the `@radix-ui/react-alert-dialog` component from radix-ui:

```bash
npm install @radix-ui/react-alert-dialog
```

2. Copy and paste the following code into your project.

<ComponentSource src="src/components/ui/AlertDialog.tsx" />

<Alert variant="outline">

This is the `<AlertDialog />` primitive. You can place it in a file at `components/ui/AlertDialog.tsx`.

</Alert>

## Usage

```tsx
import {
    AlertDialog,
    AlertDialogAction,
    AlertDialogCancel,
    AlertDialogContent,
    AlertDialogDescription,
    AlertDialogFooter,
    AlertDialogHeader,
    AlertDialogTitle,
    AlertDialogTrigger,
} from "@/components/ui/AlertDialog";
```

```tsx
<AlertDialog>
    <AlertDialogTrigger>Open</AlertDialogTrigger>
    <AlertDialogContent>
        <AlertDialogHeader>
            <AlertDialogTitle>Are you sure absolutely sure?</AlertDialogTitle>
            <AlertDialogDescription>
                This action cannot be undone. This will permanently delete your
                account and remove your data from our servers.
            </AlertDialogDescription>
        </AlertDialogHeader>
        <AlertDialogFooter>
            <AlertDialogCancel>Cancel</AlertDialogCancel>
            <AlertDialogAction>Continue</AlertDialogAction>
        </AlertDialogFooter>
    </AlertDialogContent>
</AlertDialog>
```
