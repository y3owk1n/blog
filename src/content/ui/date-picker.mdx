## Installation

1. The `<DatePicker />` component depends on the `<CalendarBase />` component. Follow the [Calendar installation instructions](/ui/calendar#installation).

2. Create the actual DatePicker component with states:

<ComponentSource src="src/components/ui/calendar/DatePicker.tsx" />

<Alert variant="outline">
    You can place it in a file at `components/ui/calendar/DatePicker.tsx`.
</Alert>

## Usage

```tsx
import { DatePicker } from "@/components/ui/calendar/Calendar";
```

```tsx
<DatePicker
    value={value}
    onChange={handleValueChange}
/>
```
