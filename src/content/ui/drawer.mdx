## Installation

1. Install the `@radix-ui/react-dialog` component from radix-ui:

```bash
npm install @radix-ui/react-dialog
```

2. Copy and paste the following code into your project.

<ComponentSource src="src/components/ui/Drawer.tsx" />

<Alert variant="outline">

This is the `<Drawer />` primitive. You can place it in a file at `components/ui/Drawer.tsx`.

</Alert>

## Usage

```tsx
import {
    Drawer,
    DrawerContent,
    DrawerDescription,
    DrawerHeader,
    DrawerTitle,
    DrawerTrigger,
} from "@/components/ui/Drawer";
```

```tsx
<Drawer>
    <DrawerTrigger>Open</DrawerTrigger>
    <DrawerContent>
        <DrawerHeader>
            <DrawerTitle>Are you sure absolutely sure?</DrawerTitle>
            <DrawerDescription>
                This action cannot be undone. This will permanently delete your
                account and remove your data from our servers.
            </DrawerDescription>
        </DrawerHeader>
    </DrawerContent>
</Drawer>
```

## Examples

### Position

Pass the `position` property to `<DrawerContent />` to indicate the edge of the screen where the component will appear. The values can be `top`, `right`, `bottom` or `left`.

### Size

Pass the `size` property to `<DrawerContent />` if you need to adjust the size of the sheet. The values can be `sm`, `default`, `lg`, `xl`, `full` or `content`.
