## Installation

### 1. Install the `@tiptap` package:

```bash
npm install @tiptap/pm @tiptap/react @tiptap/starter-kit @tiptap/extension-bubble-menu @tiptap/extension-code-block-lowlight @tiptap/extension-document @tiptap/extension-floating-menu @tiptap/extension-link @tiptap/extension-paragraph @tiptap/extension-placeholder @tiptap/extension-text @tiptap/extension-text-align @tiptap/extension-text-style @tiptap/extension-underline lowlight
```
