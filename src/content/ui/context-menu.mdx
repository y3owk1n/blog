## Installation

1. Install the `@radix-ui/react-context-menu` component from radix-ui:

```bash
npm install @radix-ui/react-context-menu
```

2. Copy and paste the following code into your project.

<ComponentSource src="src/components/ui/ContextMenu.tsx" />

<Alert variant="outline">
    This is the `<ContextMenu />` primitive. You can place it in a file at
    `components/ui/ContextMenu.tsx`.
</Alert>

## Usage

```tsx
import {
    ContextMenu,
    ContextMenuContent,
    ContextMenuItem,
    ContextMenuTrigger,
} from "@/components/ui/ContextMenu";
```

```tsx
<ContextMenu>
    <ContextMenuTrigger>Right click</ContextMenuTrigger>
    <ContextMenuContent>
        <ContextMenuItem>Profile</ContextMenuItem>
        <ContextMenuItem>Billing</ContextMenuItem>
        <ContextMenuItem>Team</ContextMenuItem>
        <ContextMenuItem>Subscription</ContextMenuItem>
    </ContextMenuContent>
</ContextMenu>
```
